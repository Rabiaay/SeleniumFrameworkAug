## Running Tests

To run tests, first clone repository using below command

```bash
git clone https://github.com/Rabiaay/SeleniumFrameworkAug.git
```

first move to project root directory 

```bash
cd SeleniumFrameworkAug
```
Then Execute below command

```bash
  mvn clean test
```
